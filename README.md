# NovaMart — Multi-Vendor E-Commerce Marketplace UI/UX

> A conversion-focused marketplace designed for real Indian shoppers 
> and vendors — built to cut cart drop-offs and make selling simple.

Author: Samikshya Mishra  
Organisation: Zidio Development  
Date: June 2026  
Version: 2.0  


## What this project is ?

NovaMart is a full UI/UX design project for a multi-vendor 
e-commerce platform serving three groups:
customers, vendors, & platform admins. The goal was to reduce cart abandonment, 
make vendor onboarding straightforward, and create a design 
system that works consistently across mobile and desktop.

## Key features designed

- Vendor onboarding — 4-step KYC wizard with progress bar
- Product discovery — filters, visual search, recommendations
- Cart and checkout — 3-step flow, address autocomplete, UPI/BNPL
- Order tracking — live status timeline with GPS map view
- Vendor dashboard — inventory, orders, analytics, ad campaigns
- Admin panel — moderation, approvals, platform analytics



## Design system


### Colours

| Name | Hex | Used for |

| Yellowish Gold | `#C9A227` |  offer Highlights,  badges |
| Pinkish Accent | `#E85C84` | promotions,lifestyle highlights |
| Dark Grey | `#2C2C2C` | Dark mode backgrounds|
| Burgundy | `#6B1E3A` | Category highlights |
|  Primary Blue| `#1A6FBF` | Buttons, CTAs, links |
| Secondary Navy | `#0D2A4E` | Headings, header/footer background |
| Chocolate Brown | `#4B2E2E` | Product card accents|
| Off‑white | `#F8F9FA` | Page backgrounds, card surfaces|


### Typography

| Level | Font | Size | Weight |
|-------|------|------|--------|
| Hero heading | Poppins | 32px | Bold |
| Section heading | Poppins | 20px | SemiBold |
| Card title | Inter | 16px | Medium |
| Body text | Inter | 14px | Regular |
| Labels | Inter | 12px | Medium |
| Captions | Inter | 11px | Regular |

### Spacing system
- Base unit: 8px
- Spacing scale: 4, 8, 12, 16, 24, 32, 48, 64px
- Card padding: 16px (mobile), 24px (desktop)
- Section padding: 24px (mobile), 48px (desktop)

## Accessibility

- WCAG 2.1 AA compliant
- Colour contrast ratio: minimum 4.5:1 for all text
- Full keyboard navigation on checkout flow
- Focus states on every interactive element

# NovaMart — Design System

## Colour Palette

| Name | Hex | Used for |
|------|-----|----------|
| Navy Blue | #0D2A4E | Headings, primary backgrounds |
| Brand Blue | #1A6FBF | Buttons, links, CTAs |
| Light Blue | #EBF3FC | Card backgrounds, info banners |
| Gold | #C9A227 | Highlights, premium badges |
| Emerald Green | #1E6B4A | Success states, discounts |
| Silver | #A9ABA8 | Secondary text, borders |
| Alert Red | #D0392B | Errors, warnings |
| Off-white | #F8F9FA | Page backgrounds |

## Typography

| Level | Font | Size | Weight |
|-------|------|------|--------|
| Hero heading | Poppins | 32px | Bold |
| Section heading | Poppins | 20px | SemiBold |
| Card title | Inter | 16px | Medium |
| Body text | Inter | 14px | Regular |
| Labels | Inter | 12px | Medium |
| Captions | Inter | 11px | Regular |

## Spacing System

Base unit: 8px
Scale: 4, 8, 12, 16, 24, 32, 48, 64px
Card padding: 16px (mobile) / 24px (desktop)
Section padding: 24px (mobile) / 48px (desktop)

## Components Built

- Navigation bar (sticky, with search and cart)
- Product card (grid and list variants)
- Filter sidebar and sort dropdown
- Cart drawer (slide-in from right)
- Checkout stepper (3 steps)
- Order tracking timeline
- Vendor onboarding wizard
- Admin data table with bulk actions
- Toast notifications
- Modal dialogs
- Rating and review component
- Empty state illustrations

## Icons

Library: Tabler Icons (outline style)
Size: 20px standard, 24px for navigation, 16px for inline
Stroke width: 1.5px throughout

## Accessibility

- Minimum contrast ratio: 4.5:1 for all body text
- 3:1 for large text and UI components
- All interactive elements have visible focus states
- ARIA labels on all icon-only buttons
- Keyboard navigation tested on checkout flow
- WCAG 2.1 AA compliant throughout
